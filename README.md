# Vet Clinic Database 

> This project contains SQL for a Vetinary clinic database.  
> It also contains SQL used to populate the database and different SQL queries made to the data entered


## Getting Started

This repository includes files with plain SQL:

- Use [schema.sql](./schema.sql) to create all tables.
- Use [data.sql](./data.sql) to populate tables with sample data.
- Check [queries.sql](./queries.sql)
